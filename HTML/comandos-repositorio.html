Criando um repositório

Depois de criar uma conta no GitHub, vamos entrar em nosso perfil e criar um repositório para enviar nossos arquivos.

Agora com o repositório criado, podemos enviar nossos arquivos. Vamos utilizar alguns comando iniciais de conexão com esse repositório.

Enviando um repositório

Vamos utilizar a sequencia de comandos a seguir:

  • git init
    para inicializar o git na pasta do projeto.
  • git commit -m "Primeiro commit"
    para inicializar o git na pasta do projeto.
  • git branch -m main
    para adicionar ao Branch principal.
  • git remote add origin https://github.com/seu-repositiry
    para se conectar ao repositório.
  • git push -u origin main
    para enviar os arquivos do projeto.
